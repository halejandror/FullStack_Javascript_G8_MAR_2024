Pasos para hacer un sándwich

  1. Toma dos rebanadas de pan.
  2. Unta mantequilla en una rebanada de pan.
  3. Agrega jamón, queso y lechuga en la otra rebanada de pan.
  4. Junta las dos rebanadas de pan para hacer el sándwich.
